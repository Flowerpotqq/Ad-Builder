# Sequence Maps

## Promotional Sequence (3 touches)
1. Launch: introduce offer and primary value
2. Reminder: reinforce value with proof
3. Last call: clear deadline and final CTA

## Welcome Sequence (4 touches)
1. Welcome + first action
2. Quick setup tutorial
3. Feature spotlight by segment
4. Success proof + upgrade path

## Newsletter Sequence (ongoing cadence)
- Weekly or monthly with fixed content modules
- Every issue includes one featured CTA

## Transactional Sequence
- Triggered event confirms status
- Follow-up only when status changes or action required

## Re-engagement Sequence (4 touches)
1. Soft check-in
2. Value update since last visit
3. Incentive or reduced-friction return path
4. Preference center or graceful sunset

## Exit criteria
- Converted: move to active lifecycle
- Unresponsive: reduce frequency or suppress
- Unsubscribed: immediate stop except required transactional notices
