# A/B Testing Playbook

## What to test first
1. Subject framing
2. Hero headline angle
3. CTA label wording
4. Offer framing (percent vs absolute value)
5. Social proof placement

## Test design rules
- Single variable tests whenever possible
- Run to meaningful sample size for each segment
- Measure by stage objective:
  - Open rate for subject tests
  - CTR for content/CTA tests
  - Conversion rate for offer tests

## Fast test matrix examples
- Promo subject:
  - A: `Early access pricing ends Friday`
  - B: `Save 20% on annual plan this week`

- CTA copy:
  - A: `Book A Free Demo`
  - B: `See It In Action`

- Hero angle:
  - A: `Never miss a lead again`
  - B: `Turn missed calls into booked jobs`

## Logging format
Record:
- audience segment
- send date/time
- variant id
- open rate
- ctr
- conversion rate
- winner and confidence
