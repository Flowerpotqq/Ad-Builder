# Personalization Rules

## Merge Variables
Recommended placeholders:
- `{{first_name}}`
- `{{company_name}}`
- `{{industry}}`
- `{{plan_name}}`
- `{{renewal_date}}`
- `{{next_step_url}}`

## Personalization Depth Levels
1. Light: name + broad segment
2. Medium: role/use-case references
3. Deep: behavior-based trigger and next-best action

## Rules
1. Use at most 2 personalized tokens in subject + preview combined.
2. Never over-personalize in first touch if relationship is cold.
3. If variable confidence is low, fallback to neutral copy.

## Segment-specific examples
- Agency owner: `Capture and respond to leads before competitors do.`
- Local services: `Turn missed calls into booked appointments.`
- Ecommerce: `Recover more carts with automated follow-up.`
