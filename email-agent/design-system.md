# Design System: Modern SaaS Email Language

Design intent: blend three proven styles without copying brand IP.
- NAP-inspired: modular card blocks, soft SaaS palette, practical service framing
- DocuSign-inspired: bold headline, minimal copy, one obvious CTA
- Wix-inspired: clean hero, generous whitespace, high-clarity onboarding energy

## Layout Blueprint (Desktop first, mobile safe)
1. Preheader strip
2. Brand bar
3. Hero block
4. Supporting proof block
5. Primary CTA block
6. Optional secondary content (feature grid or social proof)
7. Footer compliance block

## Spacing Rules
- Outer container width target: 600px
- Horizontal padding: 24px desktop, 16px mobile
- Section spacing:
  - Hero top/bottom: 28-36px
  - Content blocks: 20-28px
  - CTA margin top: 16-24px
- Line length target: 45-75 characters

## Section Hierarchy
- H1: 34-44px visual intent, but copy should be short enough for email clients
- H2: 24-30px for section pivots
- H3: 18-22px for card titles
- Body: 15-17px equivalent readability
- Meta text: 12-13px

## Button System
Primary button:
- Shape: rounded (8-999 radius based on brand style)
- Height: 44-52px visual target
- Label: 2-4 words, action-first
- Contrast ratio: pass accessibility contrast checks

Secondary button:
- Outline or text-only style
- Use only when audience needs a lower commitment path

CTA label patterns:
- Book My Demo
- Start Free Trial
- See It In Action
- Activate Offer
- Finish Setup

## Visual Balance Rules
- Text to whitespace ratio target: 40:60 for promotional and welcome emails
- Keep hero to body ratio near 35:65
- Avoid dense walls of text
- Use cards for multi-offer or multi-feature emails; max 6 cards before collapse into link list

## Card Layout Rules (NAP-style modular blocks)
- Card title max: 4 words
- Card body: 1-2 lines outcome-driven
- Consistent icon size and spacing
- Use clear category labels above card titles for scanning

## Hero Rules (Wix-style clarity)
- One strong line headline
- One support sentence
- One primary CTA
- Optional supporting visual cue line under CTA

## Minimal Promo Rules (DocuSign-style directness)
- Headline carries the offer
- One sentence context
- One CTA
- Optional urgency line (deadline, quantity, or window)

## Responsive Constraints
- First scroll should communicate:
  - what this is
  - why it matters
  - what to click
- Keep content modular so sections stack naturally on mobile.

## Color and Tone Pairing
- Professional SaaS: deep navy + light neutral + accent blue
- Friendly onboarding: bright blue + white + soft gray
- Urgency promo: dark text + high-contrast accent (orange/red only if brand allows)
- Re-engagement: calmer neutrals + contrast CTA for gentle reactivation
